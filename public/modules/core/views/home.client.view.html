ideads is a simple and fast ad network

Create ad placements in a few seconds and add them to your website with a couple lines of code.

Configure campaigns and attach ads to them easily. Choose what placements should display the campaign and ideads will match ads with places available when sizes are equal.

The dashboard shows in real time the number of ads displayed, clics and CTR (click-through rate).

Fast

Asynchronous ad loading and a blazing fast dashboard, cause time is money.

Easy

Ad placements, campaigns and ads. That's all you have to care about.

Powerful

If you need custom classes or code in your ads you can just write it, no limits.

Modern

Built with modern technologies: NodeJS, Angular and MongoDB.
